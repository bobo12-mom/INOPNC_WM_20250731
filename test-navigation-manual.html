🧪 문서함 네비게이션 수동 테스트

📋 테스트 방법

  1. 개발 서버 실행: npm run dev
  2. 브라우저에서 http://localhost:3000/auth/login 접속
  3. 테스트 계정으로 로그인: worker@inopnc.com / password123
  4. 아래 각 테스트 케이스를 순서대로 수행
  5. 각 테스트 결과를 ✅ Pass 또는 ❌ Fail로 표시

1️⃣ 홈(대시보드)에서 문서함 이동

홈 > 사이드바 "문서함" 클릭
홈 > 하단바 "문서함" 클릭
홈 > 빠른메뉴 "문서함" 클릭

2️⃣ 출력현황에서 문서함 이동

출력현황 > 사이드바 "문서함" 클릭
출력현황 > 하단바 "문서함" 클릭

3️⃣ 작업일지에서 문서함 이동

작업일지 > 사이드바 "문서함" 클릭
작업일지 > 하단바 "문서함" 클릭

4️⃣ 현장정보에서 문서함 이동

현장정보 > 사이드바 "문서함" 클릭
현장정보 > 하단바 "문서함" 클릭

5️⃣ 문서함 내부에서 다른 메뉴 이동

문서함 > 내문서함 > 사이드바 다른 메뉴들
문서함 > 내문서함 > 하단바 다른 메뉴들
문서함 > 공유문서함 > 사이드바 다른 메뉴들
문서함 > 공유문서함 > 하단바 빠른메뉴
문서함 > 공유문서함 > 하단바 출력/작업/현장
문서함 > 도면마킹 > 사이드바 다른 메뉴들
문서함 > 도면마킹 > 하단바 빠른메뉴
문서함 > 필수제출서류 > 모든 네비게이션
